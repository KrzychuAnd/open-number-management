.Responses
|===
|HTTP Code|Description|Notes

|201|Created|Resource created

|400|Bad Request|Error in input, JSON parse exception, etc.

|401|Unauthorized|Json Webtoken issue, e.g. invalid, missing, ...

|403|Forbidden|Authorized user has no access to Resource Type

|404|Client Error|Entity not found, e.g. Related Resource, Resource Status, Resource Type, ...

|412|Precondition Failed|Provided data does not meet Business Requirements, e.g. Invalid prefix or lenght of requested Resource Name

|500|Server Error|Error in system not caused by invalid user data.
